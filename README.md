## 🗂 Como instalar o projeto

```bash
  # Clonar o repositório
  $ git clone https://github.com/Educolt/frontend

  # Entrar no diretório
  $ cd frontend

  # Instalar as dependencias
  $ yarn

  # Start App
  $ yarn start

```
⚠️ Atention: Antes de rodar o app, setar o valor da porta em que à api está rodando.
